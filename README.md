# SpaceX Tribute Page

A SpaceX-themed tribute landing page built with HTML and CSS.

## Features

- Full-screen hero section with SpaceX branding
- Custom font (TheGodfather)
- Responsive image gallery
- Clean, dark-themed design inspired by SpaceX's aesthetic

## Tech Stack

- HTML5
- CSS3

## Getting Started

Open `index.html` in any browser — no build step required.
